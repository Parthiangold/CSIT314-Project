## Getting Started

### 1. Install Dependencies
Install Python Flask
```
pip install flask
```

Install SQLAlchemy
```
pip install Flask-SQLAlchemy
```

Install Argon2
```
pip install argon2-cffi
```
### 2. Run Locally
In CMD/Terminal, go to the CSIT314-Project directory, then run 'run.py'.
```
py run.py
```

The website will now be available to access at http://127.0.0.1:5000
